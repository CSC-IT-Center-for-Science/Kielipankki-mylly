PoS-tag Finnish plaintext

The plaintext PoS-tagger divides input text into sentences and the sentences into tokens and annotates the tokens with an appropriate lemma (aka base form) and a Part-of-Speech tag (actually an elaborate code also known as morpho-syntactic description) which contains a simple PoS-tag.
